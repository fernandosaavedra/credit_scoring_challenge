# Credit Scoring Challenge

Lo que se realizó es este desafío fue calcular la cantidad de precio base de un seguro para clientes de una entidad bancaria.

El análisis se dividió en 3 partes:

- Desglose de variables.
- Reducción de dimensiones en variables.
- Pruebas sobre modelos de machine learning.
- Selección de Modelo Final.
- Implementación de cálculo del valor de seguro.

Los detalles se encuentran en el notebook python llamado "credit_scoring_challenge.ipynb
